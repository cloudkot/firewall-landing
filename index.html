Windows Filtering Platform

Собственный фаервол с графиком скорости, мониторингом подключений, IDS и режимом изоляции. Разработан с использованием C#/WPF.

Скачать последнюю версию Исходный код

Интерфейс: правила, график скорости, логи

Ключевые возможности

Управление фильтрацией трафика на уровне ядра Windows через WFP.

Гибкие правила

Блокировка/разрешение по IP, порту, приложению. Время действия, импорт/экспорт JSON.

Плавный график

Входящая/исходящая скорость в реальном времени (байт/с) с возможностью остановки анализа.

IDS + Изоляция

Обнаружение сканирования портов, режим «полная изоляция» для экстренной блокировки всего трафика.

Защита паролем

Хеширование пароля (SHA‑256) и сохранение в реестре. Автозапуск с Windows.
